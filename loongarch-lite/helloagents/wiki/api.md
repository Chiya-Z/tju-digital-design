# API 手册

## 概述
本项目主要“接口”是构建与运行入口（Makefile 目标）以及 TEMU 的交互式监控器命令。

## 认证方式
不适用（本仓库无网络 API 与鉴权机制）。

---

## 接口列表

### 构建/运行（Makefile）

#### [MAKE] `TEMU/loongarch_sc`：`make`
**描述:** 编译测试程序并生成可加载镜像文件。

**输出:**
- `TEMU/inst.bin`：指令段镜像
- `TEMU/data.bin`：数据段镜像

#### [MAKE] `TEMU`：`make run`
**描述:** 编译并启动 TEMU 仿真器。

**参数:**
- `USER_PROGRAM`：可选，指定加载的用户程序参数（与 Makefile 传参保持一致）

---

### TEMU 监控器命令（交互式）

#### `help`
**描述:** 显示支持的命令列表与说明。

#### `c`
**描述:** 继续执行程序，直到结束。

#### `q`
**描述:** 退出 TEMU。
